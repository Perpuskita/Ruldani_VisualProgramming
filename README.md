## How to Install

> [!IMPORTANT]
> Untuk pengguna linux, dianjurkan menggunakan x11 karena wayland belum disupport tkinter. 
> Someday kita akan pindah ke pyqt untuk mengatasi masalah ini

### git-clone
```
git clone https://github.com/Perpuskita/Ruldani_VisualProgramming
```

```
cd ./Ruldani_VisualProgramming
```

> [!OPTIONAL]
### buat venv
```
python -m venv myenv
```

### run venv
```windows
.\myenv\Scripts\activate
```

```linux & mac
source myenv/bin/activate
```

### install requirement
```python
pip install requirement.txt
```

### install Ruldani_VisualProgramming
```python
pip install .
```
